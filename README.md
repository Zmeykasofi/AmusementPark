# СИМУЛЯТОР АТТРАКЦИОНА

Перед Вами программа-симулятор, которая умеет выбирать случайным образом человека, купившего билеты на аттракцион из очереди, и демонстрирует, сколько раз он может прокатиться на нём.

## Принцип работы программы

- Создайте список людей, введя их имена и фамилии, а также указав количество билетов на аттракцион.
- Далее программа автоматически выстроит их в очередь. 
- При использовании билета, программа выводит подобное сообщение:

```
Алексей Нетоложко (осталось билетов 2 шт.) прокатился на аттракционе.
```

- Затем человек снова добавляется в конец очереди и ждёт своей очереди прокатиться вновь и так до тех пор, пока у него не кончатся билеты.
